# Bike Sharing Prediction

Projek ini bertujuan untuk assignment project GDGoC divisi Machine Learning. Projek ini memprediksi jumlah sepeda yang disewa dengan menganalisis musim, cuaca, suhu, angin, dan kelembapan

Link Dataset : https://www.kaggle.com/datasets/lakshmi25npathi/bike-sharing-dataset

Link Streamlit : https://bikesharingprediction-irfan.streamlit.app/

**Analisis Singkat**

Berdasarkan hasil analisis, di tahun 2011 dan 2012, rata-rata jumlah penyewaan sepeda adalah 4504 sepeda, dengan jumlah terendah adalah 22 sepeda dan jumlah tertinggi 8714 sepeda. Selain itu, terjadi peningkatan atas jumlah sewa sepeda pada tahun 2012 dibandingkan dengan tahun sebelumnya. Musim Gugur adalah musim dengan tingkat sewa sepeda tertinggi, diikuti dengan musim panas, musim dingin, dan musim semi.
